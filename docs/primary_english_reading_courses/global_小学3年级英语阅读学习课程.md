## 阅读技巧
### 1. 上下文猜词
- 根据上下文猜测生词的含义
- 利用同义词、反义词、定义、例子等线索
- 注意单词的前缀和后缀
- 结合语境理解单词的具体含义

### 2. 段落大意归纳
- 识别段落的主题句
- 提取段落的主要信息
- 忽略次要信息和细节
- 用简洁的语言概括段落大意

### 3. 文章结构分析
- 识别文章的开头、中间和结尾
- 理解文章的逻辑结构
- 分析段落之间的关系
- 把握文章的整体脉络

### 4. 推理判断
- 根据文章内容进行推理
- 理解作者的隐含意图
- 分析文章的深层含义
- 做出合理的判断和结论
