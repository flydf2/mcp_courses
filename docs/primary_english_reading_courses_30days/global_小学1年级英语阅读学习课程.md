# 小学1年级英语阅读学习课程

## 课程基本信息

- **课程ID**: 1001
- **课程标题**: 小学1年级英语阅读学习课程
- **课程描述**: 专为小学1年级学生设计的英语阅读课程，通过30天的学习，培养孩子的英语阅读兴趣和基础阅读能力，每天40分钟，内容生动有趣，适合适龄小朋友。

- **课程难度**: 初级
- **课程状态**: 已发布（published）

## 课程详情

本课程专为小学1年级学生设计，全面覆盖英语阅读基础：

### 英语阅读部分
- 字母认知与发音
- 简单单词识别
- 基础句型理解
- 趣味故事阅读

## 课程特色

- 内容生动有趣，符合1年级学生认知水平
- 每天40分钟，学习节奏适中
- 注重培养良好的阅读习惯和方法
- 采用互动式学习，激发学习兴趣

### 第一章：英语阅读基础

#### 课时列表

1. **第1天：认识字母A-Z**（课时ID: 1001）
   - 类型: markdown
   - 时长: 40分钟
   - 是否免费: 是
   - 内容: 
     - 认识26个英文字母的大小写形式
     - 学习每个字母的基本发音
     - 通过生动的图片帮助记忆（如A对应apple，B对应banana等）
     - 玩字母配对游戏，巩固字母识别
     - 练习书写每个字母的正确笔顺

2. **第2天：元音字母发音**（课时ID: 1002）
   - 类型: markdown
   - 时长: 40分钟
   - 是否免费: 否
   - 内容: 
     - 学习5个元音字母（A, E, I, O, U）的基本发音
     - 每个元音字母搭配不同的辅音字母组成简单单词（如cat, bed, big, dog, sun）
     - 通过图片和发音练习巩固元音发音
     - 玩元音字母游戏，如"元音字母找朋友"
     - 练习朗读包含元音字母的单词

3. **第3天：辅音字母发音**（课时ID: 1003）
   - 类型: markdown
   - 时长: 40分钟
   - 是否免费: 否

     - 学习辅音字母的发音（如B, C, D, F, G等）
     - 每个辅音字母搭配元音字母组成简单单词（如bat, cap, dog, fan, gap）
     - 通过图片和发音练习加深理解
     - 玩辅音字母游戏，如"辅音字母接龙"
     - 练习朗读包含辅音字母的单词

4. **第4天：字母组合发音**（课时ID: 1004）
   - 类型: markdown
   - 时长: 40分钟

     - 每个字母组合扩展出多个单词（如at: cat, bat, hat; an: man, pan, can）
     - 通过图片和发音练习巩固字母组合发音
     - 玩字母组合游戏，如"字母组合拼单词"
     - 练习朗读包含字母组合的单词

5. **第5天：基础单词识别**（课时ID: 1005）

       - 颜色：red, blue, green, yellow, black

       - 动物：cat, dog, bird, fish, rabbit
     - 通过图片和发音练习记忆单词
     - 玩单词记忆游戏，如"快速认单词"

6. **第6天：单词与图片匹配**（课时ID: 1006）

     - 玩"记忆力大挑战"游戏，翻转卡片找匹配
     - 分组比赛，看谁匹配得又快又准
     - 练习用单词描述图片内容

7. **第7天：简单句子理解**（课时ID: 1007）
