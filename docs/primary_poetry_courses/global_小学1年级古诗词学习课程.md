## 诗歌原文

```
悯农
[唐] 李绅
锄禾日当午，汗滴禾下土。
谁知盘中餐，粒粒皆辛苦。
```

## 作者介绍

李绅（772年—846年），字公垂，唐代诗人。他的诗歌以反映农民生活疾苦为主题，《悯农》二首是他的代表作，其中第一首就是我们今天学习的内容。

## 诗歌解析

1. **锄禾日当午**：农民在中午太阳最毒的时候还在田里锄草。
2. **汗滴禾下土**：农民的汗水一滴一滴地滴在禾苗下面的土地上。
3. **谁知盘中餐**：有谁知道我们碗里的饭是怎么来的呢？
4. **粒粒皆辛苦**：每一粒粮食都包含着农民的辛苦劳动。

## 学习目标

1. 能够正确朗读《悯农》这首诗
2. 理解诗歌的基本含义
3. 体会农民的辛苦，培养珍惜粮食的好习惯
4. 学习诗歌中的生字词

## 教学步骤

1. **导入**：通过提问学生"我们吃的粮食是怎么来的？"引入主题
2. **朗读诗歌**：教师范读，学生跟读，分组朗读
3. **理解诗歌**：逐句讲解诗歌含义，结合图片帮助理解
4. **互动讨论**：讨论为什么要珍惜粮食，我们应该怎么做
5. **背诵练习**：通过游戏方式帮助学生背诵诗歌
6. **总结**：总结诗歌的主题和学习收获

## 练习题目

2. 问答：

   （2）诗歌告诉我们什么道理？

3. 实践：
   回家后帮爸爸妈妈做一件家务，体会劳动的辛苦。

## 诗歌原文

```
静夜思
[唐] 李白
床前明月光，疑是地上霜。
举头望明月，低头思故乡。
```

## 作者介绍

李白（701年—762年），字太白，号青莲居士，唐代伟大的浪漫主义诗人，被后人称为"诗仙"。他的诗歌风格豪放、飘逸，充满想象力。

## 诗歌解析

1. **床前明月光**：明亮的月光照在床前。
2. **疑是地上霜**：诗人一开始以为是地上的霜。
3. **举头望明月**：抬头看着天上的明月。
4. **低头思故乡**：低下头思念远方的家乡。

## 学习目标

1. 能够正确朗读《静夜思》这首诗
2. 理解诗歌的基本含义
3. 感受诗人的思乡之情
4. 培养学生的想象力和表达能力

## 教学步骤

1. **导入**：通过展示月亮的图片，引导学生想象在月光下的感受
2. **朗读诗歌**：教师范读，学生跟读，个人朗读
3. **理解诗歌**：逐句讲解诗歌含义，结合情境帮助理解
4. **想象练习**：让学生想象自己在月光下的感受，用自己的话表达
5. **背诵练习**：通过比赛方式帮助学生背诵诗歌
6. **总结**：总结诗歌的主题和学习收获

## 诗歌原文

```
春晓
[唐] 孟浩然
春眠不觉晓，处处闻啼鸟。
夜来风雨声，花落知多少。
```

## 作者介绍

孟浩然（689年—740年），字浩然，号孟山人，唐代著名的山水田园诗人。他的诗歌风格清淡、自然，善于描绘山水田园风光。

## 诗歌解析

1. **春眠不觉晓**：春天睡觉，不知不觉天就亮了。
2. **处处闻啼鸟**：到处都能听到鸟儿的叫声。
3. **夜来风雨声**：昨天夜里听到了风雨的声音。
4. **花落知多少**：不知道有多少花朵被风雨吹落了。

## 学习目标

1. 能够正确朗读《春晓》这首诗
2. 理解诗歌的基本含义
3. 感受春天的美好景色
4. 培养学生观察自然的能力

## 教学步骤

1. **导入**：通过展示春天的图片，引导学生感受春天的美好
2. **朗读诗歌**：教师范读，学生跟读，小组朗读
3. **理解诗歌**：逐句讲解诗歌含义，结合春天的景象帮助理解
4. **观察练习**：让学生说一说自己观察到的春天的变化
5. **背诵练习**：通过唱歌方式帮助学生背诵诗歌
6. **总结**：总结诗歌的主题和学习收获

## 诗歌原文

```
咏鹅
[唐] 骆宾王
鹅，鹅，鹅，曲项向天歌。
白毛浮绿水，红掌拨清波。
```

## 作者介绍

骆宾王（约619年—约687年），字观光，唐代诗人。他七岁时就写下了《咏鹅》这首诗，被称为"神童"。他的诗歌风格清新、自然，富有童趣。

## 诗歌解析

1. **鹅，鹅，鹅**：连续三个"鹅"字，生动地表现了鹅的叫声。
2. **曲项向天歌**：鹅弯曲着脖子，对着天空唱歌。
3. **白毛浮绿水**：白色的羽毛漂浮在绿色的水面上。
4. **红掌拨清波**：红色的脚掌划动着清澈的水波。

## 学习目标

1. 能够正确朗读《咏鹅》这首诗
2. 理解诗歌的基本含义
3. 感受诗歌的生动形象
4. 培养学生的语言表达能力和观察能力

## 教学步骤

1. **导入**：通过展示鹅的图片，引导学生观察鹅的外形特征
2. **朗读诗歌**：教师范读，学生跟读，个人朗读
3. **理解诗歌**：逐句讲解诗歌含义，结合鹅的形象帮助理解
4. **表演练习**：让学生模仿鹅的动作，加深对诗歌的理解
5. **背诵练习**：通过游戏方式帮助学生背诵诗歌
6. **总结**：总结诗歌的主题和学习收获
